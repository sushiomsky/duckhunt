About This Test

This page simulates the duck hunt game to test the Chrome extension's ability to:

  • Detect duck elements on the page
  • Calculate correct positions accounting for scroll offset
  • Handle ducks appearing at different viewport positions
  • Work correctly when the page is scrolled

Expected Behavior

When a duck appears (after clicking spawn buttons), the extension should:

  1. Detect the duck element within 100ms
  2. Log the detection to the console
  3. Calculate the duck's position correctly (regardless of scroll)
  4. Automatically "click" on the duck
  5. The duck should disappear or show some feedback

Bug Fix Verification

Original Bug: Ducks would appear to fly at the bottom of the page after scrolling.

Fix: The getAbsolutePosition() function now properly calculates positions using window.scrollY/scrollX with fallbacks, ensuring correct positioning regardless of scroll position.

To Verify Fix:

  1. Scroll down the page significantly
  2. Spawn a duck at the top, middle, or bottom
  3. Check console logs to verify correct position calculation
  4. Verify the duck is clicked successfully
